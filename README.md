Sample CRUD App
=============

In the future:
Online tool for tracking and analyzing technology trends. This project helps me to learn new technologies. 

Now:
Simple CRUD for now - managing keywords to track.


#### Tools, technologies: ####

Server side:
  * ASP.NET MVC + Web API,
  * MS SQL Server,
  * Entity Framework.

Client side:
  * AngularJS.

Testing:
  * Selenium (+PhantomJS),
  * Jasmine,
  * MSTest.

Other:
  * TeamCity (for CI).
