TrendsTracker
=============

Online tool for tracking and analyzing technology trends. 

The project is still in early development phase.

#### Tools, technologies: ####

Server side:
  * ASP.NET MVC + Web API
  * SQL Server 
  * Entity Framework (6)

Client side:
  * AngularJS

Testing:
  * Selenium (+PhantomJS)

Other:
  * TeamCity (for CI)
  * Jira (for Scrum)
