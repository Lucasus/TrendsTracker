TrendsTracker
=============

Online tool for tracking and analyzing technology trends. 

The project is still in early development phase.

#### Tools, technologies: ####

Server side:
  * ASP.NET MVC + Web API,
  * SQL Server ,
  * Entity Framework.

Client side:
  * AngularJS.

Testing:
  * Selenium (+PhantomJS),
  * MSTest + NUnit (yes, both).

Other:
  * TeamCity (for CI),
  * Jira (for Scrum).
